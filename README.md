**Overview**

This project is a .NET and React-based expense tracker designed to help users manage both expenses and incomes effectively. The application provides a complete workflow for organizing transactions, maintaining expense and income groups, viewing dashboard summaries, and supporting additional premium-style features for an improved user experience.

**Main Features**

**Dashboard**

Last week summary chart
Last 5 expense changes with graphical comparison
Last 5 income changes with graphical comparison
Navigation menus for expense and income groups
Additional UI sections like testimonials, FAQ, newsletter, and footer links

**Income and Expense Management**

Add, edit, and delete income or expense entries
Table view with fields such as Id, Name, Amount, and Group
Search functionality
Filtering by min amount, max amount, and group
Sorting options
Pagination and row count controls
Bulk selection and bulk delete
Export stats through email

**Income and Expense Group Management**

Manage groups with name and description
Add, edit, and delete groups
Paginated and sortable tables
Clickable group details page
Group-level recent account changes view

**Premium Features**

Reminders managed through profile settings
Blog creation and blog detail pages
Dashboard reminder visibility
Blog cards and details such as title, author, description, body, and created date

**Integrations**

Pipeline status email notifications for failed pipelines
Mailchimp integration for newsletter registration during sign-up
